# **App Name**: AbyssBuilds

## Core Features:

- Build Display: Show a database of user-generated builds using a card-based layout. Includes character portraits, build names, creator info, and basic stats.
- Filtering & Sorting: Allow users to filter and sort builds by character, element, weapon type, content focus, and more. Sorting options include popularity, recent updates, and ratings.
- Detailed Build View: Provide a detailed view for each build, showcasing character art, stats, Demon Wedge configuration, weapon loadout, combat stats, playstyle strategies, and team compositions.
- Build Creator: An interactive build creator/editor with character selection, Demon Wedge grid, weapon selection, real-time stat calculations, and tolerance tracking.
- Character & Weapon Database: Pages for browsing characters and weapons with detailed information including skills, stats, recommended builds, and lore.
- Team Builder: A team composition tool for theory-crafting. Displays synergy scores between party members and generates an overview for different possible rotations.

## Style Guidelines:

- Primary background: Very dark purple (#0a0612) to set a dark fantasy tone.
- Secondary background: Dark purple-gray (#1a1525) for depth.
- accent primary: Vibrant purple (#a855f7) for main interactive elements.
- Headers: 'Poppins', a bold and elegant sans-serif font for a modern feel.
- Body Text: 'Inter', a clean sans-serif font for readability.
- Use custom icons that clearly communicate element types (fire, wind, water, etc.), weapon types, and roles.
- Subtle hover effects, loading animations, and smooth transitions to enhance user experience.
