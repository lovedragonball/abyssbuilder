# คู่มือการใช้งานระบบแจ้งเตือนการ Craft

## ฟีเจอร์หลัก

### 1. ตั้งเวลาแจ้งเตือนสำหรับการ Craft
- เปิดรายละเอียดของไอเทมที่สามารถ craft ได้ (Upgrade Materials หรือ Secret Letters)
- กำหนดจำนวนที่ต้องการ craft
- กดปุ่ม "Start Crafting Timer" เพื่อเริ่มนับเวลา
- ระบบจะแจ้งเตือนอัตโนมัติเมื่อครบเวลา

### 2. การแจ้งเตือนผ่าน Browser
- ครั้งแรกที่ใช้งาน ระบบจะขออนุญาตแสดงการแจ้งเตือน
- กดปุ่ม "Enable Notifications" เพื่ออนุญาต
- เมื่อครบเวลา จะมีการแจ้งเตือนผ่าน browser notification
- การแจ้งเตือนจะแสดงแม้ว่าจะเปิดแท็บอื่นอยู่

### 3. ติดตามสถานะการ Craft
- ดูสถานะการ craft ทั้งหมดที่กำลังดำเนินการได้ที่ด้านบนของหน้า
- แสดงเวลาที่เหลือแบบ real-time
- สามารถยกเลิกการแจ้งเตือนได้ตลอดเวลา

### 4. ไอคอนแสดงสถานะ
- ไอเทมที่กำลัง craft จะมีไอคอน Timer แสดงอยู่
- แสดงเวลาที่เหลือในโหมด list view
- Badge แสดงจำนวนการ craft ที่กำลังดำเนินการ

## วิธีใช้งาน

### เริ่มต้นใช้งาน
1. ไปที่หน้า Materials & Forging
2. เลือกไอเทมที่ต้องการ craft (เฉพาะไอเทมที่มีสูตร craft)
3. กำหนดจำนวนที่ต้องการ
4. กดปุ่ม "Start Crafting Timer"

### การอนุญาตการแจ้งเตือน
1. เมื่อกดเริ่ม timer ครั้งแรก จะมีข้อความขออนุญาต
2. กดปุ่ม "Enable Notifications"
3. เลือก "Allow" ในหน้าต่างที่เบราว์เซอร์แสดง

### การจัดการ Timer
- **ดู Timer ทั้งหมด**: ดูได้ที่ส่วน "Active Crafting Timers" ด้านบนหน้า
- **ยกเลิก Timer**: กดปุ่ม X ที่มุมบนขวาของแต่ละ timer
- **ดูรายละเอียด**: กดที่ไอเทมเพื่อเปิดรายละเอียดและดูสถานะ

## ข้อมูลเพิ่มเติม

### ไอเทมที่รองรับ
- **Upgrade Materials**: Repulsive Crystal, Gold Sand, Projectile, Sharpening Potion
- **Secret Letters**: อาวุธทุกชนิด (Melee และ Ranged)

### การเก็บข้อมูล
- ข้อมูล timer จะถูกเก็บใน localStorage
- แม้ปิดเบราว์เซอร์แล้วเปิดใหม่ timer ยังคงทำงานต่อ
- ข้อมูลจะถูกลบอัตโนมัติหลังจากครบเวลา 1 นาที

### เคล็ดลับ
- ใช้ฟีเจอร์ Pin เพื่อติดตามไอเทมที่ต้องการ craft บ่อยๆ
- ตั้ง timer หลายๆ อันพร้อมกันได้
- ดูเวลารวมและต้นทุนรวมก่อนเริ่ม craft

## การแก้ปัญหา

### ไม่ได้รับการแจ้งเตือน
1. ตรวจสอบว่าได้อนุญาตการแจ้งเตือนแล้ว
2. ตรวจสอบการตั้งค่าเบราว์เซอร์
3. ลองรีเฟรชหน้าและตั้ง timer ใหม่

### Timer หายไป
- ตรวจสอบว่าไม่ได้ล้าง localStorage
- ตรวจสอบว่าไม่ได้เปิดในโหมด Incognito

### การแจ้งเตือนไม่ทำงาน
- ตรวจสอบว่าเบราว์เซอร์รองรับ Notification API
- ลองเปิดใช้งานการแจ้งเตือนใหม่ในการตั้งค่าเบราว์เซอร์
